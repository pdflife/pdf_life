## Website for PDF file manipulation with Python
you can: 
- remove pages
- cut pages
- merge pages
- extract
- PDF to JPG
- rotate
- move
- watermark
- invert
- extract images
- add blank pages
- combine pages

## Run

Run the file: main.py

or
access [PDFLife](https://pdflife.onrender.com/)

Code is written in JavaScript and Python using Flask.
